Project Overview (English Version)

This project provides a real-time analysis of cryptocurrencies using data collected from the CoinGecko API.
A dataset of 250 cryptocurrencies is retrieved in real-time, including key indicators such as:
• id
• current_price
• market_cap
• price_change_percentage_24h
• ath (All-Time High)
• atl (All-Time Low)
• timestamps

Based on these 250 rows, the project automatically generates two datasets:
	1.	Top 10 Positive Cryptocurrencies
Sorted by the highest price_change_percentage_24h in real-time.
	2.	Top 10 Negative Cryptocurrencies
Sorted by the lowest price_change_percentage_24h in real-time.

These datasets allow quick insight into market performance trends, helping identify the strongest and weakest assets within the last 24 hours.

Ce projet propose une analyse en temps réel des cryptomonnaies grâce aux données récupérées depuis l’API CoinGecko.
Un dataset de 250 cryptomonnaies est extrait en temps réel, incluant les indicateurs suivants :
• id
• current_price
• market_cap
• price_change_percentage_24h
• ath (plus haut historique)
• atl (plus bas historique)
• timestamps

À partir de ces 250 lignes, le projet génère automatiquement deux datasets :
	1.	Top 10 des Cryptomonnaies les Plus Positives
Triées par price_change_percentage_24h décroissant.
	2.	Top 10 des Cryptomonnaies les Plus Négatives
Triées par price_change_percentage_24h croissant.

Ces datasets permettent d’identifier rapidement les monnaies les plus performantes et les moins performantes des dernières 24h.
